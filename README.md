

remote LCD

## methdology
- Server
 raspberry pi control LED on/off via SMBus controller

- client
android phone
send remote control command to Server

## how to run it?
- for server part
% cd raspberrypi
% python server.py

- for client part
using eclipse to import prj and port exe to your android device
